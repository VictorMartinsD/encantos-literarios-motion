## Sobre o Projeto

O **Encantos Literários** é uma landing page desenvolvida para um clube de assinatura de livros. O foco principal deste projeto foi a aplicação de conceitos avançados de CSS e microinterações para criar uma experiência imersiva:

- **Animações Fluidas:** Uso de transições coordenadas para a revelação de itens do kit mensal e elementos decorativos.
- **Ambient Glow:** Implementação de gradientes elípticos (`radial-gradient`) para criar efeitos de luz ambiente que não interferem na legibilidade de outras seções.
- **Estabilidade Visual:** Tratamento rigoroso de estados de `hover` para evitar deslocamentos de layout (layout shifts), garantindo uma interface polida.
- **Responsividade:** Design adaptável para diferentes tamanhos de tela, do mobile ao desktop.

# Encantos Literários - Motion (EN)

## About the Project

**Encantos Literários** is a landing page developed for a book subscription club. The main focus of this project was the application of advanced CSS concepts and micro-interactions to create an immersive experience:

- **Fluid Animations:** Use of coordinated transitions to reveal monthly kit items and decorative elements.
- **Ambient Glow:** Implementation of elliptical gradients (`radial-gradient`) to create ambient lighting effects that do not interfere with the readability of other sections.
- **Visual Stability:** Strict handling of `hover` states to prevent layout shifts, ensuring a polished interface.
- **Responsiveness:** Adaptive design for different screen sizes, from mobile to desktop.
