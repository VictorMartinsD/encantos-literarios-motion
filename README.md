## Sobre o Projeto

O **Encantos Literários** é uma landing page de um clube de assinatura que explora o poder do CSS moderno para criar uma narrativa visual. O projeto foca em microinterações e performance, garantindo que animações complexas não prejudiquem a experiência do usuário.

## Tecnologias e Conceitos

Para alcançar o resultado visual e técnico, utilizei:

- **CSS Fan-out & Choreography:** Implementação de animações coordenadas usando `@keyframes` e `cubic-bezier` para o efeito de "leque" nos cards de planos.
- **JavaScript Trigger Optimization:** Uso estratégico de JavaScript para manipulação de classes DOM. Utilizei a propriedade `{ once: true }` no listener de evento para garantir que o gatilho da animação seja disparado apenas uma vez, otimizando o consumo de memória e processamento.
- **SVG Sprites:** Uso eficiente de ícones através da tag `<use>`, reduzindo requisições HTTP.
- **Ambient Glow Effects:** Aplicação de `radial-gradient` elípticos para criar profundidade visual.
- **Mobile First & Responsividade:** Estrutura desenvolvida inicialmente para dispositivos móveis (Mobile First).
- **Conventional Commits:** Histórico de versionamento organizado e profissional.

# Encantos Literários - Motion (EN)

## About the Project

**Encantos Literários** is a book subscription club landing page that explores the power of modern CSS to create a visual narrative. The project focuses on micro-interactions and performance, ensuring that complex animations do not hinder the user experience.

## Technologies and Concepts

To achieve the visual and technical results, I used:

- **CSS Fan-out & Choreography:** Coordinated animations using `@keyframes` and `cubic-bezier` for the "fan" effect on plan cards.
- **JavaScript Trigger Optimization:** Strategic use of JavaScript for DOM class manipulation. I used the `{ once: true }` property in the event listener to ensure the animation trigger fires only once, optimizing memory and processing.
- **SVG Sprites:** Efficient icon usage via the `<use>` tag, reducing HTTP requests.
- **Ambient Glow Effects:** Application of elliptical `radial-gradient` to create visual depth.
- **Mobile First & Responsiveness:** Structure initially developed for mobile devices (Mobile First).
- **Conventional Commits:** Organized and professional versioning history.
